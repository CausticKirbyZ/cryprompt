# cryprompt

A crystal library for creating easy prompting cli tools in crystal 

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     cryprompt:
       github: CausticKirbyZ/cryprompt
   ```

2. Run `shards install`
